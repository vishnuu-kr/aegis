[ 02 ]The five primitives

Everything a human uses online. Governed.

Capability is defined at the primitive level — not per-site. A browser, an inbox, a number, money, compute. Any website works on day one.

Browser 01

Headful Playwright automation. Credentials are injected into the page — the agent never sees the secret.

Email 02

Its own inbox. OTPs and magic links resolve autonomously — the verification loop closes without you.

Phone 03

A programmable number. SMS 2FA and TOTP codes handled on demand — the seed never leaves the Vault.

Money 04

Virtual cards with network-enforced spend limits. A hijacked agent physically cannot overspend.

Compute 05

Sandboxed Firecracker microVMs and its own cloud org. The agent builds, deploys, and pays the bill — zero human taps.

[ 04 ]Policy engine

One gate between the agent and the world.

Every action passes through the policy engine. A pure function the agent can't argue with — it reads the mandate and returns exactly one verdict.

ALLOW Routine — execute and log.
NOTICE High-salience — log loudly, notify.
STEP-UP Needs a signed human approval.
DENY No mandate — refuse and log.

The Passport

Every agent gets a cryptographic DID — an Ed25519 key that signs its requests, every audit entry, and is the subject of every mandate. Revoke it once and all authority starves.

agent passport 
{
  "id": "did:key:z6Mk…AGENT",
  "operator": "did:key:z6Mk…HUMAN",
  "key": "Ed25519",
  "status": "active",
  "proof": { /* operator sig */ }
}
The Mandate

A human-signed, scoped, expiring grant of authority. Spend caps, allowed merchants, time bounds and the approval line — all expressed as fields. Tune autonomy by editing a mandate, never code.

pay mandate 
{
  "capability": "pay",
  "limitPerTxn": 50.00,
  "period": "P30D",
  "stepUpAbove": 100.00,
  "expires": "2026-12-31"
}
