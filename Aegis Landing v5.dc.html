06How it works

Three commands. Full governance.

From zero to a governed agent in under a minute. Every step produces cryptographic artifacts you can verify offline.

01

Mint a passport

Create a cryptographic identity for your agent in one command.

$ agentgrid passport create --agent "Research Agent"
✓ did:key:z6MkuYt9hP8n...AGENT
02

Sign a mandate

Define exactly what the agent can do — spend limits, allowed domains, tool access, session duration.

{ "trust_domain": "*.stripe.com", "session_limit": "$100.00", "expires": "24h" }
03

Deploy & govern

The agent operates within its mandate. Every action logged. Every anomaly flagged. Revoke with one key.

$ agentgrid revoke --agent Research
✓ Passport revoked · all sessions terminated
