03The tool surface

One MCP server. Eight tools.

Drop AgentGrid into Claude or any MCP-compatible agent. The agent sees only these tools - behind each one sits the policy engine, the vault, and the audit ledger. It can't reach a capability except through a tool, and no tool acts without a verdict.

agentgrid · tools 
browse(action, target)         // drive the browser
comms.send(channel, to, body)  // email / sms
comms.verify(sess)             // read OTP / link
pay(amount, merchant, card?)   // virtual card
deploy(project, files, env)    // build & host
provision(kind)                // new inbox/phone/card
vault.use(handle, ...)         // use, never read
approve(action, ctx)           // ask the human

A human-signed, scoped, expiring grant of authority. Spend caps, allowed merchants, time bounds and the approval line - all expressed as fields. Tune autonomy by editing a mandate, never code.
